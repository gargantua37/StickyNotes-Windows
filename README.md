## üìñ How to Use

1.  **Run the App:** Double-click `StickyNotes.exe`.
2.  **Add Note:** Type a title and content, then click **Add**.
3.  **Find Note:** Type in the Search box.
4.  **Manage Notes:**
    *   **Right-Click** on a note in the list (right panel) to open it.
    *   **Right-Click** on the sticky note itself to **Edit** content.
    *   **Double-Click** on the sticky note to **Close/Hide** it.
